## ✨ Highlights

- 🌀 **3D Immersion**: High-fidelity 3D scenes using `React Three Fiber` and `Three.js`.
- 🎬 **Cinematic Animations**: Complex, physics-based animations with `Motion` and `Framer Motion`.
- 🌌 **Visual Effects**: Advanced post-processing for a unique aesthetic feel.
- 🛠️ **Modern Stack**: Built with the latest tools including `Next.js 16` and `React 19`.

## 🚀 Spin Up Locally

Get the portfolio running on your machine in minutes.

### 1. Prerequisites

- **Node.js** (v18.x+)
- **npm** or **yarn**

### 2. Setup

```bash
# Clone the repository
git clone https://github.com/AMZNGM/ngm.git

# Navigate to the project directory
cd ngm

# Install dependencies
npm install
```

### 3. Development

```bash
npm run dev
```

The app will be available at [http://localhost:3000](http://localhost:3000).
